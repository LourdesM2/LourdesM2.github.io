HTML Reference

< !DOCTYPE html >

It defines the document type
The DTD specifies the rules for the markup language, so that the browsers render the content correctly.
Tip: Always add the <!DOCTYPE> declaration to your HTML documents, so that the browser knows what type of document to expect.
For example:
<!DOCTYPE html>
<html>

<!--...-->

Defines a comment
The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.

For Example:
<!--This is a comment. Comments are not displayed in the browser--> <This is a paragraph.>

< a >

It defines a hyperlink.
The <a> tag defines a hyperlink, which is used to link from one page to another.
For Example: <a href="https://www.w3schools.com">Visit W3Schools.com! >

< abbr>

Defines an abbreviation or an acronym
The <abbr> tag defines an abbreviation or an acronym, like "Mr.", "Dec.", "ASAP", "ATM". Tip:An abbreviation and an acronym are both shortened versions of something else. Both are often represented as a series of letters .
For example:The <abbr title="World Health Organization">WHO <abbr > was founded in 1948.

< head >

Defines information about the document
The following elements can go inside the <head> element:

  • <title>
    • (this element is required in an HTML document)
  • <style>
  • base>
  • <link>>
  • <meta>>
  • <script>>
  • <noscript>>

< header >

It defines a header for a document or section <br> A <header> element typically contains:

  • one or more heading elements (<h1> - <h6>)
  • logo or icon
  • authorship information

< img >

It defines a image <
The <img> tag defines an image in an HTML page.
The <img> tag has two required attributes: src and alt.
Note: Images are not technically inserted into an HTML page, images are linked to HTML pages. The <img> tag creates a holding space for the referenced image.
Tip: To link an image to another document, simply nest the <img> tag inside <a> tags.

< li >

It defines a list item
The <li> tag defines a list item.
The <li> tag is used in ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).
For Example:
<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

< meta >

Defines metadata about an HTML document
Metadata is data (information) about data.
The <meta> tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable.
Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.
The metadata can be used by browsers (how to display content or reload page), search engines (keywords), or other web services.

< p >

Defines a paragraph
The <p> tag defines a paragraph. Browsers automatically add some space (margin) before and after each <p> element. The margins can be modified with CSS (with the margin properties).

For Example:
<p>This is some text in a paragraph.</p>

< style >

Defines style information for a document
The <style> tag is used to define style information for an HTML document.
Inside the <tyle> element you specify how HTML elements should render in a browser. For Example:
<html>
<head>
<style>
h1 {color:red;}
p {color:blue;}
</style>
</head>
<body>

<h1>A heading</h1>
<p>A paragraph.</p>

</body>
</html>

< ul >

Defines an unordered list
The <ul> tag defines an unordered (bulleted) list.
Use the <ul> tag together with the <li> tag to create unordered lists.
For Example:
<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

< sub >

Defines subscripted text
The <sub> tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O. Tip: Use the <sup> tag to define superscripted text.
Use the <ul> tag together with the <li> tag to create unordered lists.
For Example:
<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

< audio >

It defines a sound content.
The <audio> tag defines sound, such as music or other audio streams.
Currently, there are 3 supported file formats for the <audio> element: MP3, Wav, and Ogg:\

For Example:
<audio controls> <source src="horse.ogg" type="audio/ogg"> <source src="horse.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio>

< bdo >

bdo stands for Bi-Directional Override.
The <bdo> tag is used to override the current text direction.
For Example:
<bdo dir="rtl"> This text will go right-to-left. </bdo>

< blockquote >

It defines a section that is quoted from another source
Browsers usually indent <blockquote> elements.
For Example:
<blockquote cite="http://www.worldwildlife.org/who/index.html"> For 50 years, WWF has been protecting the future of nature.
The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
</blockquote>

< iframe >

An inline frame is used to embed another document within the current HTML document.
For Example:
<iframe src="https://www.w3schools.com">
</iframe>

< input >

Defines an input control
<input> elements are used within a <form> element to declare input controls that allow users to input data.
An input field can vary in many ways, depending on the type attribute.
For Example:
<form action="/action_page.php"> First name: <input type="text" name="fname"><br> Last name: <input type="text" name="lname"><br> <input type="submit" value="Submit"> </form>
